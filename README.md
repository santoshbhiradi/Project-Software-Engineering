
Requirements

Install the following before running the project:

- Node.js
- npm
- VS Code (recommended)
- Live Server extension (VS Code)

How to Run the Project:

1. Open project folder
     Open the project in VS Code.

2. Install dependencies
     Run in terminal:
       npm install

3. Run the code
    node main/backend/server.js (Make sure the terminal is run to correct directory)

Local host appears: http://localhost:3000
